Полное руководство по CSS для начинающих

CSS (Cascading Style Sheets — каскадные таблицы стилей) — это язык, который отвечает за внешний вид сайта. Если HTML — это скелет, то CSS — это кожа, одежда и макияж. С помощью CSS мы задаем цвета, размеры, расположение элементов, анимацию и многое другое.

Важно: CSS можно подключать тремя способами:
  • Внешний файл: <link rel="stylesheet" href="styles.css">
  • Внутренний стиль: <style> ... </style> в секции <head>
  • Инлайн-стили: атрибут style="..." прямо в HTML-теге

Уровень 1: Основы синтаксиса

1. Структура CSS-правила

Каждое CSS-правило состоит из двух частей: селектора (к кому применяем) и блока объявлений (что меняем).

селектор {
свойство1: значение1;
свойство2: значение2;
}

Пример:

p {
color: red;
font-size: 16px;
}

Этот код говорит: "Найди все параграфы <p> и сделай их текст красным (color: red), а размер шрифта — 16 пикселей (font-size: 16px)".

2. Способы подключения CSS

Внешний файл (рекомендуемый способ):

<head>
<link rel="stylesheet" href="style.css">
</head>

Внутренний стиль:

<head>
<style>
p { color: blue; }
</style>
</head>

Инлайн-стили (используйте редко):

<p style="color: green; font-size: 20px;">Зеленый текст</p>

Уровень 2: Селекторы (учимся выбирать элементы)

3. Базовые селекторы

Селектор по тегу (элементу): выбирает все элементы с указанным тегом.

p { /* все параграфы */
color: blue;
}

h1 { /* все заголовки h1 */
text-align: center;
}

Селектор по классу (class): выбирает элементы с определенным классом. Начинается с точки ..

.warning { /* все элементы с class="warning" */
color: orange;
font-weight: bold;
}

Селектор по идентификатору (id): выбирает элемент с конкретным id. Начинается с решетки #. Помните: id должен быть уникальным!

#header { /* элемент с id="header" */
background-color: black;
color: white;
}

Универсальный селектор (*): выбирает все элементы на странице.

* {
margin: 0; /* убирает отступы у всех элементов */
padding: 0;
}

4. Комбинированные селекторы

Селектор потомка (пробел): выбирает элементы внутри других элементов.

div p { /* все <p> внутри <div> */
margin-left: 20px;
}

Дочерний селектор (>): выбирает только прямых детей.

ul > li { /* только <li>, которые прямые дети <ul> */
list-style-type: square;
}

Соседний селектор (+): выбирает элемент, который идет сразу после другого.

h2 + p { /* параграф, который идет сразу после h2 */
font-style: italic;
}

Селектор атрибута: выбирает элементы по наличию или значению атрибута.

input[type="text"] { /* все текстовые поля ввода */
border: 1px solid blue;
}

a[target="_blank"] { /* ссылки, открывающиеся в новом окне */
color: red;
}

5. Группировка селекторов

Если нескольким селекторам нужны одинаковые стили, их можно перечислить через запятую.

h1, h2, h3 {
font-family: Arial, sans-serif;
color: navy;
}

Уровень 3: Свойства и значения (работа с внешностью)

6. Работа с текстом

Основные свойства текста:

  • color — цвет текста (можно указывать в hex, rgb, rgba, названиях)
  • font-size — размер шрифта (px, em, rem, %)
  • font-family — шрифт
  • font-weight — жирность (normal, bold, 100-900)
  • font-style — наклон (normal, italic)
  • text-align — выравнивание (left, center, right, justify)
  • text-decoration — подчеркивание (none, underline, line-through)
  • line-height — межстрочный интервал
p {
color: #333;
font-size: 16px;
font-family: Arial, Helvetica, sans-serif;
font-weight: normal;
text-align: left;
line-height: 1.5;
}

h1 {
color: rgb(0, 100, 200);
text-decoration: underline;
font-weight: bold;
}

7. Цвета и фон

Способы задания цвета:

  • Имена: red, blue, green и др.
  • HEX (шестнадцатеричный): #ff0000 (красный), #00ff00 (зеленый)
  • RGB: rgb(255, 0, 0)
  • RGBA (с прозрачностью): rgba(255, 0, 0, 0.5) (полупрозрачный красный)

Свойства фона:

  • background-color — цвет фона
  • background-image — фоновое изображение
  • background-repeat — повторение фона
  • background-position — позиция фона
  • background-size — размер фона
body {
background-color: #f0f0f0;
}

div {
background-color: rgba(0, 150, 0, 0.2); /* полупрозрачный зеленый */
background-image: url('pattern.png');
background-repeat: repeat-x;
}

8. Блочная модель (Box Model)

Каждый элемент в CSS — это прямоугольный блок, который состоит из:

  • Content — содержимое (текст, картинка)
  • Padding — внутренний отступ (от содержимого до границы)
  • Border — граница (рамка)
  • Margin — внешний отступ (от границы до соседних элементов)
div {
width: 300px; /* ширина содержимого */
height: 200px; /* высота содержимого */
padding: 20px; /* внутренний отступ со всех сторон */
border: 2px solid red; /* рамка */
margin: 30px; /* внешний отступ со всех сторон */
}

Можно задавать отступы по отдельности:

  • padding-top, padding-right, padding-bottom, padding-left
  • margin-top, margin-right, margin-bottom, margin-left

Сокращенная запись:

/* 4 значения: верх, право, низ, лево */
padding: 10px 20px 15px 25px;

/* 3 значения: верх, право/лево, низ */
margin: 10px 20px 15px;

/* 2 значения: верх/низ, право/лево */
padding: 10px 20px;

/* 1 значение: все стороны */
margin: 10px;

9. Рамки (borders)

border: 1px solid black; /* толщина, стиль, цвет */

/* Можно разделять */
border-width: 2px;
border-style: dashed; /* solid, dotted, dashed, double, groove, ridge */
border-color: blue;

/* Только одна сторона */
border-bottom: 3px dotted red;
border-left: 5px solid green;

Скругление углов:

border-radius: 10px; /* все углы */
border-radius: 50%; /* круг (для квадратного элемента) */
border-radius: 10px 20px; /* верхние левый/правый, нижние */

Уровень 4: Позиционирование и Layout

10. Свойство display

display определяет, как элемент ведет себя на странице.

  • display: block — блочный элемент (занимает всю ширину)
  • display: inline — строчный элемент (ширина по содержимому)
  • display: inline-block — строчный, но можно задавать ширину/высоту
  • display: none — элемент скрыт и не занимает места
  • display: flex — гибкий контейнер (flexbox)
  • display: grid — сетка (grid layout)
/* Превращаем ссылки в блоки */
a {
display: block;
width: 200px;
padding: 10px;
background-color: #007bff;
color: white;
}

11. Позиционирование (position)

Свойство position определяет метод позиционирования элемента.

  • position: static — по умолчанию, обычный поток документа
  • position: relative — относительно своего обычного места
  • position: absolute — относительно ближайшего позиционированного родителя
  • position: fixed — относительно окна браузера (не двигается при скролле)
  • position: sticky — прилипает при скролле

Для позиционирования используются свойства: top, right, bottom, left, z-index.

.relative {
position: relative;
top: 20px; /* сместится вниз на 20px от обычного места */
left: 30px; /* сместится вправо на 30px */
}

.absolute {
position: absolute;
top: 0;
right: 0; /* прижмется к правому верхнему углу родителя */
}

.fixed {
position: fixed;
bottom: 20px;
right: 20px; /* кнопка в правом нижнем углу экрана */
}

12. Flexbox (гибкая раскладка)

Flexbox — это современный способ расположения элементов в строку или колонку. Идеален для одномерных макетов.

Родитель (flex-контейнер):

.container {
display: flex; /* включаем flexbox */
flex-direction: row; /* направление: row (по умолчанию) или column */
justify-content: center; /* выравнивание по главной оси (горизонтально) */
align-items: center; /* выравнивание по поперечной оси (вертикально) */
gap: 20px; /* расстояние между элементами */
flex-wrap: wrap; /* перенос на новую строку */
}

Значения justify-content: flex-start, flex-end, center, space-between, space-around, space-evenly

Значения align-items: flex-start, flex-end, center, stretch, baseline

Дети (flex-элементы):

.item {
flex: 1; /* элемент растягивается */
align-self: center; /* переопределить выравнивание для конкретного элемента */
order: 2; /* порядок элемента (по умолчанию 0) */
}

Пример центрирования:

.parent {
display: flex;
justify-content: center;
align-items: center;
height: 100vh; /* высота на весь экран */
}

13. Grid (сетка)

CSS Grid — мощная система для двумерных макетов (строки и колонки одновременно).

.grid-container {
display: grid;
grid-template-columns: 1fr 2fr 1fr; /* три колонки: вторая в 2 раза шире */
grid-template-rows: 100px auto 200px; /* три строки */
gap: 20px; /* отступы между ячейками */
}

Единицы измерения в Grid:

  • px — пиксели
  • % — проценты
  • fr — дробная единица (flexible fraction)
  • auto — по содержимому
  • minmax(100px, 1fr) — от минимума до максимума

Размещение элементов:

.item1 {
grid-column: 1 / 3; /* занимает колонки с 1 по 3 */
grid-row: 1 / 2; /* занимает первую строку */
}

.item2 {
grid-column: span 2; /* растянуться на 2 колонки */
}

Уровень 5: Адаптивность и медиа-запросы

14. Медиа-запросы (Media Queries)

Медиа-запросы позволяют применять разные стили в зависимости от характеристик устройства (ширина экрана, ориентация и т.д.).

/* Базовые стили для мобильных */
body { font-size: 14px; }

/* Для планшетов (ширина от 768px) */
@media (min-width: 768px) {
body { font-size: 16px; }
}

/* Для десктопов (ширина от 1200px) */
@media (min-width: 1200px) {
body { font-size: 18px; }
}

Другие условия:

  • max-width — максимальная ширина
  • orientation: portrait — книжная ориентация
  • orientation: landscape — альбомная ориентация
  • hover: hover — устройство с поддержкой наведения
/* Для устройств с мышью (не для телефонов) */
@media (hover: hover) {
button:hover {
background-color: darkblue;
}
}

/* Альбомная ориентация */
@media (orientation: landscape) {
.sidebar {
width: 30%;
}
}

15. Адаптивная верстка

Принципы адаптивной верстки:

  • Используйте относительные единицы (%, vw, vh, em, rem) вместо пикселей
  • Применяйте медиа-запросы для разных устройств
  • Делайте изображения гибкими

Гибкие изображения:

img {
max-width: 100%; /* изображение не вылезет за родителя */
height: auto; /* сохраняет пропорции */
}

Относительные единицы:

  • vw — 1% от ширины окна (viewport width)
  • vh — 1% от высоты окна
  • em — относительно размера шрифта родителя
  • rem — относительно размера шрифта корневого элемента (html)
.fullscreen {
width: 100vw; /* на всю ширину окна */
height: 100vh; /* на всю высоту окна */
}

html { font-size: 16px; }
h1 { font-size: 2rem; } /* 32px (16px * 2) */
p { font-size: 1rem; } /* 16px */

Уровень 6: Продвинутые темы

16. Псевдоклассы и псевдоэлементы

Псевдоклассы (:) — определяют состояние элемента.

/* Ссылки */
a:link { color: blue; } /* непосещенная ссылка */
a:visited { color: purple; } /* посещенная ссылка */
a:hover { color: red; } /* при наведении мыши */
a:active { color: green; } /* в момент клика */

/* Формы */
input:focus {
border-color: blue;
outline: none;
}

input:disabled {
background-color: #eee;
}

/* Структурные */
li:first-child { font-weight: bold; } /* первый элемент */
li:last-child { border-bottom: none; } /* последний элемент */
li:nth-child(odd) { background: #f0f0f0; } /* нечетные элементы */
li:nth-child(3) { color: red; } /* третий элемент */

Псевдоэлементы (::) — создают виртуальные элементы.

/* Первая буква или строка */
p::first-letter {
font-size: 3em;
color: red;
}

p::first-line {
font-weight: bold;
}

/* Добавление контента до и после */
a::before {
content: "→ ";
}

a::after {
content: " ←";
}

/* Для очистки float */
.clearfix::after {
content: "";
display: table;
clear: both;
}

17. Анимации и трансформации

Трансформации (transform):

.transform {
transform: rotate(45deg); /* поворот */
transform: scale(1.5); /* увеличение */
transform: translate(50px, 20px); /* перемещение */
transform: skew(10deg); /* наклон */
/* можно комбинировать */
transform: rotate(45deg) scale(1.2);
}

Переходы (transitions): плавное изменение свойств.

.button {
background-color: blue;
transition: background-color 0.3s ease;
}

.button:hover {
background-color: darkblue;
}

Параметры transition:

  • transition-property — что меняем
  • transition-duration — длительность
  • transition-timing-function — функция времени (ease, linear, ease-in, ease-out)
  • transition-delay — задержка
.box {
width: 100px;
height: 100px;
background: red;
transition: width 0.5s ease, height 1s linear 0.2s;
}

.box:hover {
width: 200px;
height: 200px;
}

Ключевые кадры (keyframes) — сложная анимация:

@keyframes slidein {
from {
transform: translateX(-100%);
opacity: 0;
}
to {
transform: translateX(0);
opacity: 1;
}
}

@keyframes bounce {
0% { transform: translateY(0); }
50% { transform: translateY(-50px); }
100% { transform: translateY(0); }
}

.animated {
animation-name: bounce;
animation-duration: 1s;
animation-iteration-count: infinite;
animation-timing-function: ease;
}

Сокращенная запись анимации:

.element {
animation: bounce 1s ease infinite;
}

18. Переменные в CSS (Custom Properties)

CSS-переменные позволяют хранить значения для повторного использования.

:root {
--main-color: #007bff;
--secondary-color: #6c757d;
--font-size: 16px;
--spacing: 20px;
}

.button {
background-color: var(--main-color);
color: white;
padding: var(--spacing);
font-size: var(--font-size);
}

.alert {
border: 2px solid var(--main-color);
margin: var(--spacing);
}

Переменные с запасным значением:

.text {
color: var(--text-color, black); /* если --text-color не задан, будет black */
}

19. Единицы измерения и специфичность

Специфичность (важность селекторов):

Когда несколько правил применяются к одному элементу, побеждает самое специфичное:

  1. Инлайн-стили (самые сильные)
  2. ID (#header)
  3. Классы, атрибуты, псевдоклассы (.menu, :hover)
  4. Теги и псевдоэлементы (p, ::before) (самые слабые)

Важность (!important): переопределяет всё, но используйте редко!

p {
color: red !important; /* это перебьет даже инлайн-стили */
}

Бонус-трек: Практические примеры

20. Типовые задачи

Центрирование элемента:

/* Горизонтально */
.block {
margin: 0 auto;
width: 300px;
}

/* Вертикально и горизонтально (flex) */
.parent {
display: flex;
justify-content: center;
align-items: center;
height: 100vh;
}

Трехколоночный макет:

.container {
display: grid;
grid-template-columns: 1fr 2fr 1fr;
gap: 20px;
}

@media (max-width: 768px) {
.container {
grid-template-columns: 1fr;
}
}

Карточка товара с hover-эффектом:

.card {
width: 250px;
border: 1px solid #ddd;
border-radius: 8px;
padding: 15px;
transition: box-shadow 0.3s ease;
}

.card:hover {
box-shadow: 0 4px 15px rgba(0,0,0,0.2);
}

.card img {
width: 100%;
height: auto;
transition: transform 0.3s ease;
}

.card:hover img {
transform: scale(1.05);
}

Что дальше?

Поздравляем! Вы изучили основы CSS. Теперь вы можете:

  • Изучить препроцессоры (Sass, Less) — они делают CSS еще удобнее
  • Освоить CSS-фреймворки (Bootstrap, Tailwind) — готовые решения
  • Изучить продвинутые техники анимации
  • Практиковаться на реальных макетах

Помните: лучший способ выучить CSS — верстать реальные проекты. Начните с простых макетов и постепенно усложняйте задачи. Удачи!

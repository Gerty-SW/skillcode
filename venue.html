Полное руководство по JavaScript для начинающих

JavaScript (JS) — это язык программирования, который делает веб-страницы живыми. Если HTML — это скелет, CSS — одежда, то JavaScript — это мышцы и мозг. С помощью JS мы можем реагировать на действия пользователя, изменять содержимое страницы, отправлять запросы на сервер и многое другое.

Важно: JavaScript не имеет никакого отношения к языку Java. Это разные языки, которые объединяет только похожее название.

Как подключить JavaScript к HTML:

  • Внешний файл: <script src="script.js"></script>
  • Внутренний скрипт: <script> ... код ... </script>
Рекомендация: Размещайте тег <script> перед закрывающим тегом </body>, чтобы скрипты не блокировали загрузку страницы.

Уровень 1: Основы синтаксиса

1. Переменные и константы

Переменные — это контейнеры для хранения данных. В JavaScript есть три способа объявления переменных:

  • let — переменная, которую можно изменять (современный способ)
  • const — константа, значение нельзя изменить
  • var — устаревший способ (лучше не использовать)
let name = "Анна"; // строковая переменная
let age = 25; // числовая переменная
const PI = 3.14159; // константа (нельзя изменить)
let isStudent = true; // логическое значение (boolean)

// Изменяем значение переменной
age = 26; // OK
// PI = 3.14; // Ошибка! const нельзя изменять

Правила именования переменных:

  • Имена могут содержать буквы, цифры, символы $ и _
  • Имя не может начинаться с цифры
  • Регистр имеет значение: myName и myname — разные переменные
  • Используйте camelCase для составных имен: userName, firstName

2. Типы данных

В JavaScript есть 8 основных типов данных. Вот самые важные для начинающих:

Примитивные типы:

  • number — числа (целые и дробные)
  • string — строки (текст)
  • boolean — логические значения (true/false)
  • undefined — значение не определено
  • null — "ничего", пустое значение

Ссылочные типы:

  • object — объекты (сложные структуры)
  • array — массивы (списки данных)
let number = 42; // number
let float = 3.14; // тоже number
let text = "Привет, мир!"; // string
let isTrue = true; // boolean
let notDefined; // undefined
let empty = null; // null

// Проверка типа данных
console.log(typeof number); // "number"
console.log(typeof text); // "string"

3. Базовые операторы

Арифметические операторы:

let a = 10;
let b = 3;

console.log(a + b); // 13 (сложение)
console.log(a - b); // 7 (вычитание)
console.log(a * b); // 30 (умножение)
console.log(a / b); // 3.333... (деление)
console.log(a % b); // 1 (остаток от деления)
console.log(a ** b); // 1000 (возведение в степень 10³)

Операторы сравнения:

console.log(5 > 3); // true (больше)
console.log(5 < 3); // false (меньше)
console.log(5 >= 5); // true (больше или равно)
console.log(5 == "5"); // true (нестрогое равенство, сравнивает значения)
console.log(5 === "5"); // false (строгое равенство, сравнивает типы)
console.log(5 != "5"); // false (нестрогое неравенство)
console.log(5 !== "5"); // true (строгое неравенство)
Совет: Всегда используйте строгое сравнение === и !==, чтобы избежать неожиданных ошибок.

Логические операторы:

let isLogged = true;
let isAdmin = false;

console.log(isLogged && isAdmin); // false (И — оба должны быть true)
console.log(isLogged || isAdmin); // true (ИЛИ — хотя бы один true)
console.log(!isLogged); // false (НЕ — отрицание)

4. Взаимодействие с пользователем

Простые способы общения с пользователем:

// Вывод сообщения
alert("Привет, мир!");

// Запрос подтверждения
let isAdult = confirm("Вам есть 18 лет?");
console.log(isAdult); // true (если нажали OK) или false (если Отмена)

// Запрос ввода текста
let name = prompt("Как вас зовут?", "Гость");
console.log("Привет, " + name);

// Вывод в консоль (для разработчиков)
console.log("Это сообщение видно в консоли браузера");

Уровень 2: Управляющие конструкции

5. Условные операторы

if...else — выполняет код в зависимости от условия.

let age = 17;

if (age >= 18) {
console.log("Вы совершеннолетний");
} else {
console.log("Вы несовершеннолетний");
}

else if — несколько условий.

let score = 85;

if (score >= 90) {
console.log("Отлично!");
} else if (score >= 75) {
console.log("Хорошо");
} else if (score >= 60) {
console.log("Удовлетворительно");
} else {
console.log("Нужно подучить");
}

Тернарный оператор — сокращенная запись if...else.

let age = 20;
let status = age >= 18 ? "взрослый" : "ребенок";
console.log(status); // "взрослый"

switch — для множества конкретных значений.

let day = 3;
let dayName;

switch (day) {
case 1:
dayName = "Понедельник";
break;
case 2:
dayName = "Вторник";
break;
case 3:
dayName = "Среда";
break;
default:
dayName = "Неизвестный день";
}
console.log(dayName); // "Среда"

6. Циклы

for — когда знаем количество повторений.

// Вывод чисел от 1 до 5
for (let i = 1; i <= 5; i++) {
console.log(i); // 1, 2, 3, 4, 5
}

// i++ то же самое, что i = i + 1
// Можно уменьшать: i--

while — когда условие проверяется перед выполнением.

let i = 1;
while (i <= 5) {
console.log(i);
i++;
}

do...while — выполнится хотя бы один раз (условие проверяется после).

let j = 1;
do {
console.log(j);
j++;
} while (j <= 5);

break и continue — управление циклом.

// break - прерывает цикл
for (let i = 1; i <= 10; i++) {
if (i === 5) {
break; // остановится на 5
}
console.log(i); // 1, 2, 3, 4
}

// continue - пропускает итерацию
for (let i = 1; i <= 5; i++) {
if (i === 3) {
continue; // пропускаем 3
}
console.log(i); // 1, 2, 4, 5
}

Уровень 3: Функции

7. Объявление функций

Функции — это блоки кода, которые можно вызывать многократно.

Function Declaration (объявление функции):

function greet(name) {
return "Привет, " + name + "!";
}

// Вызов функции
let message = greet("Анна");
console.log(message); // "Привет, Анна!"

Function Expression (функциональное выражение):

const greet = function(name) {
return "Привет, " + name + "!";
};

console.log(greet("Иван")); // "Привет, Иван!"

Стрелочные функции (современный способ):

const greet = (name) => {
return "Привет, " + name + "!";
};

// Если функция возвращает значение сразу, можно сократить:
const greetShort = name => "Привет, " + name + "!";

// Если параметров несколько, нужны скобки:
const sum = (a, b) => a + b;

console.log(greet("Мария")); // "Привет, Мария!"
console.log(sum(5, 3)); // 8

8. Параметры и аргументы

Параметры по умолчанию:

function greet(name = "Гость") {
return "Привет, " + name + "!";
}

console.log(greet()); // "Привет, Гость!"
console.log(greet("Анна")); // "Привет, Анна!"

Rest-параметры (...): собирают оставшиеся аргументы в массив.

function sumAll(...numbers) {
let total = 0;
for (let num of numbers) {
total += num;
}
return total;
}

console.log(sumAll(1, 2, 3)); // 6
console.log(sumAll(10, 20, 30, 40)); // 100

Уровень 4: Массивы

9. Создание и базовые операции

Массив — это упорядоченный список данных. Индексация начинается с 0.

// Создание массивов
let fruits = ["яблоко", "банан", "апельсин"];
let numbers = [1, 2, 3, 4, 5];
let mixed = ["текст", 42, true, null]; // можно смешивать типы

// Доступ к элементам
console.log(fruits[0]); // "яблоко" (первый элемент)
console.log(fruits[2]); // "апельсин" (третий элемент)
console.log(fruits.length); // 3 (длина массива)
console.log(fruits[fruits.length - 1]); // последний элемент

// Изменение элементов
fruits[1] = "груша";
console.log(fruits); // ["яблоко", "груша", "апельсин"]

10. Методы массивов

Добавление и удаление элементов:

let fruits = ["яблоко", "банан"];

// push - добавить в конец
fruits.push("апельсин");
console.log(fruits); // ["яблоко", "банан", "апельсин"]

// pop - удалить с конца
let last = fruits.pop();
console.log(last); // "апельсин"
console.log(fruits); // ["яблоко", "банан"]

// unshift - добавить в начало
fruits.unshift("груша");
console.log(fruits); // ["груша", "яблоко", "банан"]

// shift - удалить с начала
let first = fruits.shift();
console.log(first); // "груша"
console.log(fruits); // ["яблоко", "банан"]

Перебор массивов:

let fruits = ["яблоко", "банан", "апельсин"];

// Цикл for
for (let i = 0; i < fruits.length; i++) {
console.log(fruits[i]);
}

// for...of (современный способ)
for (let fruit of fruits) {
console.log(fruit);
}

// forEach
fruits.forEach(function(fruit, index) {
console.log(index + ": " + fruit);
});

// Стрелочная функция с forEach
fruits.forEach((fruit, index) => {
console.log(`${index}: ${fruit}`);
});

Полезные методы:

let numbers = [1, 5, 2, 8, 3];

// indexOf - поиск индекса элемента
console.log(numbers.indexOf(5)); // 1
console.log(numbers.indexOf(10)); // -1 (не найден)

// includes - проверка наличия
console.log(numbers.includes(8)); // true
console.log(numbers.includes(7)); // false

// slice - создает копию части массива
let sliced = numbers.slice(1, 3); // с индекса 1 до 3 (не включая)
console.log(sliced); // [5, 2]

// splice - изменяет массив (удаляет/добавляет)
let removed = numbers.splice(2, 1); // удалить 1 элемент с индекса 2
console.log(removed); // [2]
console.log(numbers); // [1, 5, 8, 3]

// join - объединяет в строку
let str = fruits.join(", ");
console.log(str); // "яблоко, банан, апельсин"

11. Трансформация массивов

map — создает новый массив, преобразуя каждый элемент.

let numbers = [1, 2, 3, 4];

let doubled = numbers.map(num => num * 2);
console.log(doubled); // [2, 4, 6, 8]

let strings = numbers.map(num => "Число: " + num);
console.log(strings); // ["Число: 1", "Число: 2", "Число: 3", "Число: 4"]

filter — создает новый массив с элементами, прошедшими проверку.

let numbers = [1, 2, 3, 4, 5, 6];

let even = numbers.filter(num => num % 2 === 0);
console.log(even); // [2, 4, 6] (только четные)

let big = numbers.filter(num => num > 3);
console.log(big); // [4, 5, 6]

reduce — сворачивает массив в одно значение.

let numbers = [1, 2, 3, 4];

let sum = numbers.reduce((accumulator, current) => accumulator + current, 0);
console.log(sum); // 10

// accumulator - накопленный результат
// current - текущий элемент
// 0 - начальное значение аккумулятора

sort — сортирует массив.

let fruits = ["банан", "яблоко", "апельсин"];
fruits.sort();
console.log(fruits); // ["апельсин", "банан", "яблоко"] (по алфавиту)

let numbers = [4, 2, 10, 1];
numbers.sort((a, b) => a - b); // для чисел
console.log(numbers); // [1, 2, 4, 10]

Уровень 5: Объекты

12. Создание объектов

Объекты хранят данные в формате "ключ-значение".

// Создание объекта
let user = {
name: "Анна",
age: 25,
isAdmin: true,
"likes cats": true // если ключ с пробелом, нужны кавычки
};

// Доступ к свойствам
console.log(user.name); // "Анна" (точечная нотация)
console.log(user["age"]); // 25 (скобочная нотация)
console.log(user["likes cats"]); // true (только скобочная)

// Изменение свойств
user.age = 26;
user.city = "Москва"; // добавление нового свойства

// Удаление свойств
delete user.isAdmin;

console.log(user);

13. Методы объектов

Методы — это функции, которые являются свойствами объекта.

let user = {
name: "Анна",
age: 25,
// Метод (старый синтаксис)
sayHi: function() {
console.log("Привет, я " + this.name);
},
// Современный синтаксис методов
sayAge() {
console.log("Мне " + this.age + " лет");
}
};

user.sayHi(); // "Привет, я Анна"
user.sayAge(); // "Мне 25 лет"

Ключевое слово this ссылается на текущий объект.

14. Перебор объектов

let user = {
name: "Анна",
age: 25,
city: "Москва"
};

// for...in - перебор ключей
for (let key in user) {
console.log(key); // name, age, city
console.log(user[key]); // Анна, 25, Москва
}

// Object.keys() - получаем массив ключей
let keys = Object.keys(user);
console.log(keys); // ["name", "age", "city"]

// Object.values() - массив значений
let values = Object.values(user);
console.log(values); // ["Анна", 25, "Москва"]

// Object.entries() - массив пар [ключ, значение]
let entries = Object.entries(user);
console.log(entries); // [["name", "Анна"], ["age", 25], ["city", "Москва"]]

15. Копирование и ссылки

Важно: Объекты хранятся по ссылке.

let user1 = { name: "Анна" };
let user2 = user1; // копируется ссылка, а не объект

user2.name = "Мария";
console.log(user1.name); // "Мария" - изменилось и в user1!

// Как правильно копировать объекты
let user3 = { name: "Анна", age: 25 };

// Способ 1: Object.assign
let copy1 = Object.assign({}, user3);

// Способ 2: spread-оператор (современный)
let copy2 = { ...user3 };

copy2.name = "Мария";
console.log(user3.name); // "Анна" - не изменилось!

Уровень 6: Продвинутые концепции

16. Шаблонные строки

Шаблонные строки (обратные кавычки) позволяют вставлять переменные в строку.

let name = "Анна";
let age = 25;

// Старый способ
console.log("Меня зовут " + name + ", мне " + age + " лет");

// Новый способ (шаблонные строки)
console.log(`Меня зовут ${name}, мне ${age} лет`);

// Можно выполнять выражения
console.log(`Через 5 лет мне будет ${age + 5} лет`);

// Многострочные строки
let text = `Первая строка
Вторая строка
Третья строка`;

17. Деструктуризация

Деструктуризация позволяет удобно извлекать значения из массивов и объектов.

Деструктуризация массивов:

let fruits = ["яблоко", "банан", "апельсин"];

// Старый способ
let first = fruits[0];
let second = fruits[1];

// Деструктуризация
let [a, b, c] = fruits;
console.log(a); // "яблоко"
console.log(b); // "банан"
console.log(c); // "апельсин"

// Можно пропускать элементы
let [firstFruit, , thirdFruit] = fruits;
console.log(firstFruit); // "яблоко"
console.log(thirdFruit); // "апельсин"

// Значения по умолчанию
let [x, y, z, w = "виноград"] = fruits;
console.log(w); // "виноград" (значение по умолчанию)

Деструктуризация объектов:

let user = {
name: "Анна",
age: 25,
city: "Москва"
};

// Деструктуризация
let { name, age, city } = user;
console.log(name); // "Анна"
console.log(age); // 25

// Переименование переменных
let { name: userName, age: userAge } = user;
console.log(userName); // "Анна"

// Значения по умолчанию
let { country = "Россия" } = user;
console.log(country); // "Россия"

18. Spread и Rest операторы

Spread (...) — разворачивает массив или объект.

// Spread для массивов
let arr1 = [1, 2, 3];
let arr2 = [4, 5, 6];

let combined = [...arr1, ...arr2];
console.log(combined); // [1, 2, 3, 4, 5, 6]

// Копирование массива
let copy = [...arr1];

// Spread для объектов
let user = { name: "Анна", age: 25 };
let withCity = { ...user, city: "Москва" };
console.log(withCity); // { name: "Анна", age: 25, city: "Москва" }

// Spread в вызовах функций
function sum(a, b, c) {
return a + b + c;
}
let numbers = [1, 2, 3];
console.log(sum(...numbers)); // 6

Rest (...) — собирает оставшиеся аргументы.

// Rest с массивами
let [first, ...rest] = [1, 2, 3, 4, 5];
console.log(first); // 1
console.log(rest); // [2, 3, 4, 5]

// Rest с объектами
let { name, ...info } = { name: "Анна", age: 25, city: "Москва" };
console.log(name); // "Анна"
console.log(info); // { age: 25, city: "Москва" }

Бонус-трек: Работа с DOM

19. Поиск элементов

DOM (Document Object Model) — это представление HTML-страницы в виде объектов, с которыми можно работать из JavaScript.

// Поиск по ID
let header = document.getElementById('header');

// Поиск по классу
let items = document.getElementsByClassName('item');

// Поиск по тегу
let paragraphs = document.getElementsByTagName('p');

// Современные методы (рекомендуется)
let element = document.querySelector('.menu'); // первый элемент
let elements = document.querySelectorAll('.item'); // все элементы

20. Изменение содержимого

// Изменение текста
element.textContent = "Новый текст";

// Изменение HTML
element.innerHTML = "<strong>Жирный текст</strong>";

// Изменение стилей
element.style.color = "red";
element.style.fontSize = "20px";

// Добавление/удаление классов
element.classList.add('active');
element.classList.remove('hidden');
element.classList.toggle('visible'); // переключает класс

21. Обработка событий

// Навешиваем обработчик события
let button = document.querySelector('#myButton');

button.addEventListener('click', function() {
alert('Кнопка нажата!');
});

// Со стрелочной функцией
button.addEventListener('click', () => {
console.log('Клик!');
});

// Другие события
element.addEventListener('mouseover', handler); // наведение мыши
element.addEventListener('mouseout', handler); // уход мыши
input.addEventListener('input', handler); // ввод текста
window.addEventListener('scroll', handler); // скролл

JavaScript — это не страшно, это просто требует практики!
